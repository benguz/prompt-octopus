This is a small, open source project under the Apache 2.0 license, included in the project's Github. A more thorough terms of service is still a work in progress.

This site uses Plausible Analytics, an open source, privacy-friendly analytics provider. We don't collect personal information, just pageviews. No cookies.

The only code not "visible" on this site is the endpoint that handles API requests for the non-self hosted version, and is nearly identical to the flask function in /python_selfhost. It is hosted on pythonanywhere.com. If you know of an easy way for me to add transparency to it, do let me know at guzovsky@princeton.edu.

Designing a safe infrastructure for your API keys is a work in progress. When they live in your localStorage, they're vulnerable to XSS attacks, and there may be other risks I'm not aware of. Self-hosting is a safer option. Also consider creating API keys with strict budget/usage limits in your OpenAI console.
